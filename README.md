# WeatherApp
WeatherApp — это простое приложение для прогноза погоды в Москве, которое использует API (https://api.weatherapi.com) для получения данных прогноза по часам и на несколько дней.

## Libraries and Tools
- ViewModel
- ViewBinding
- Picasso
- Volley
